# ML-Homeworks: Full Pipeline from EDA to Clustering & PCA


## 📖 Описание 📖

Этот репозиторий содержит серию учебных заданий по машинному обучению — от первичного анализа данных до методов кластеризации и снижения размерности.  
Каждая работа оформлена как отдельный ноутбук или скрипт с выводами, визуализациями и метриками качества.

**Стек технологий:**  
`Python`, `pandas`, `numpy`, `matplotlib`, `seaborn`, `plotly`, `scikit-learn`, `SciPy`, `XGBoost`, `LightGBM`, `CatBoost`, `SHAP`
