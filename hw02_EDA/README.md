# ДЗ №2: EDA Titanic

## Описание
Разведочный анализ данных датасета Titanic: от первичного обзора до feature engineering.

## Структура
- `hw02_EDA.ipynb` — ноутбук с полным анализом
- `requirements.txt` — зависимости
- `README.md` — описание

## Этапы
✅ Быстрый обзор данных  
✅ Обработка пропусков (медиана по классу для Age, мода для Embarked)  
✅ Расширенная статистика (min, max, mean, median, skewness, kurtosis)  
✅ Feature Engineering (One-Hot, Target Encoding, Feature Hashing, новые признаки)  
✅ Визуализация (matplotlib, seaborn, plotly)  
✅ Итоговые выводы и гипотезы

```bash
pip install -r requirements.txt
jupyter notebook hw02_EDA.ipynb
