# Исследование линейной регрессии: предсказание стоимости автомобилей

## Описание проекта
Данный проект представляет собой комплексное исследование по построению модели линейной регрессии для предсказания стоимости автомобилей. В работе проведен полный цикл разработки модели машинного обучения: от анализа данных до сравнения различных моделей с регуляризацией.

## Структура проекта
- `hw04_LogisticRegression.ipynb` - Jupyter Notebook с полным исследованием
- `README.md` - описание проекта
- `ToyotaCorolla.csv` - датасет

## [Датасет](https://www.kaggle.com/datasets/vishakhdapat/price-of-used-toyota-corolla-cars/data)
Использован датасет CarPrice_Assignment.csv, содержащий информацию об автомобилях:
- **Целевая переменная**: price (цена автомобиля)
- **Признаки**: технические параметры (двигатель, топливная система, размеры) и описательные характеристики (марка, тип кузова, привод)

### 1. Разделение данных
- Соотношение: 70% обучающая, 30% тестовая выборка
- Обоснование: тестовая выборка имитирует "невидимые" данные для объективной оценки качества

### 2. Обученные модели
1. **LinearRegression** - базовая модель без регуляризации
2. **Ridge** - L2 регуляризация
3. **Lasso** - L1 регуляризация

Для Ridge и Lasso проведен подбор гиперпараметра alpha с помощью GridSearchCV.

### 3. Оценка качества
**Использованные метрики:**
- **RMSE** (Root Mean Square Error): чувствительна к большим ошибкам, интерпретируема в исходных единицах
- **MAE** (Mean Absolute Error): устойчива к выбросам, простая интерпретация
- **R²** (коэффициент детерминации): показывает долю объясненной дисперсии

## Использованные библиотеки
- pandas, numpy - обработка данных
- matplotlib, seaborn - визуализация
- scikit-learn - модели и метрики
